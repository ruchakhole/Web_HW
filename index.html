Summary: Latitude vs. X


The project was meant to analyze weather changes closer to the equator. To accomplish this analysis,data was pulled from the OpenWeatherMap API to assemble a dataset on over 500 cities.


After assembling the dataset, Matplotlib was used to plot various aspects of the weather vs. latitude. Factors studied include: cloudiness, windspeed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends obeserved.
